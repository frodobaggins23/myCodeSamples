Komu patří tahle stránka?

Jednomu potrhlému klukovi, který se rozhodl, že si splní sen a naučí se stavět webové stránky. Tenhle web je jeho úplně první pokus, jeho první krůček na dlouhé cestě. Najdete zde všechno, co se zatím naučil.

Výzva 1 - HTML/CSS

HMTL/CSS Mojí první výzvou je zvládnout taje HTML a CSS. Chtěl bych zvládnout postavit web od A až do Z, od wireframu, přes design až po HTML.

Výzva 2 - Javascript

Odtud je už krůček k opravdovému programování. Nejprve je třeba zkrotit konzoli a pak už půjde všechno samo.
